NewEvent

data class NewEvent(summary: EventSummary?, location: EventLocation?, endDate: EventDate, alldayEvent: EventAllday?, startDate: EventDate?, recurrence: EventRecurrence?, description: EventDescription?, url: EventUrl?)
Content copied to clipboard

An event to be created.

Note that end date values are mandatory.

Parameters

summary

the event's summary, which is shown as its title

location

the location associated with this event

alldayEvent

whether or not, this event spans, one or more whole days

startDate

the event's start date

endDate

the event's end date

recurrence

for recurring events, conforms to RFC 2445

description

the event's description, which is shown as its notes

url

the url associated with the event
