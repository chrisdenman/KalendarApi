data class Event(summary: EventSummary, location: EventLocation, alldayEvent: EventAllday, startDate: EventDate, endDate: EventDate, recurrence: EventRecurrence, excludedDates: List<EventDate>, status: EventStatus, description: EventDescription, url: EventUrl, identifier: Identifier, sequence: EventSequence, stampDate: EventDate)
Content copied to clipboard

A preexisting event.

Parameters

summary

the event's summary, which is shown as its title

location

the location associated with this event

alldayEvent

whether or not, this event spans, one or more whole days

startDate

the event's start date

endDate

the event's end date

recurrence

for recurring events, conforms to RFC 2445

excludedDates

a list of excluded dates, if this is a recurring event

status

the event's participant confirmation status, only applies if the event has participants

description

the event's description, which is shown as its notes

url

the url associated with the event

identifier

the unique identifier assigned when the event was made

sequence

a versioning sequence-number

stampDate

the event's last modification date
